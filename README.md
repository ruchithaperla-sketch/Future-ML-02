# Customer Support Ticket Classification (NLP)

##  Overview
This project builds a Machine Learning model that automatically classifies customer support tickets based on ticket descriptions using NLP techniques and Naive Bayes algorithm.

## Objective
- Classify customer support tickets automatically
- Reduce manual work
- Improve response time

##  Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF
- Naive Bayes
- Streamlit

## 📂 Dataset
Customer Support Ticket Dataset (Kaggle)

- Ticket Description → Text
- Ticket Type → Category

## ⚙️ Workflow
1. Load dataset  
2. Clean text  
3. TF-IDF vectorization  
4. Train Naive Bayes model  
5. Predict category  
6. Streamlit UI
